# Leo's Numpad #

This numpad features 16 keys and volume/playback control through a rotary encoder.

## Features ##
- 16 keys arranged in a 4x4 matrix
- Rotary encoder for volume control
- Rotary encoder doubles as a play/pause button
- 3 Piece 3D Printed Case

## PCB ##
Designed in KiCad. 
The switches are arranged in a 4x4 column to row matrix, with an extra column added for the switch on rotary encoder.
